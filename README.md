# Projeto de Banco de Dados

O projeto consiste na criação de um banco para solucionar uma determinada problemática, assim como de uma API para realizar o CRUD nesse banco.

## Tecnologias utilizadas

- Laravel
- PHP
- MySQL
- Instância EC2
- Swagger
